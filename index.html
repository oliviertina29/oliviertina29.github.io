Academic Projects

Regression in massive data analysis

This was the subject of my final degree project. This research internship allowed me to get into the world of data and to understand regression and other data analysis methods such as PCA.

Extreme Values Distributions

In this project, we learnt how to find extreme values of a given dataset trough Block Maxima or Peak Over Treshold, define the distribution of those extreme values and then build tools such us Value At Risk (VaR) and Expected Shortfall (ES) to help in business decisions making. Finally, we applied Extreme values distribution on Meta Stock.

Analysing the Casablanca Stock Exchange data

We downloaded financial data from Casa Stock Exchange for some Banks such as Bank of Africa, Attijariwafa Bank, ...Then, we convert data from .aspx to .csv. We have essentially created a class to compute mean, max, momentum, moving average and visualization.

Wave Equation Simulation

In this project, we begin by some theoretical stuff such as consistancy, order and stability then we write python code to simulate the wave equation in 1D and 2D. This project helped us to have have a deep understanding of Finite Difference Method.
